## Initial setup
Run Ethereum client. Ganache recommended. 

```bash
npm i
truffle compile
truffle migrate
```
## Running the app

```bash
npm start
```
App will automagically open in a new tab in your browser on http://localhost:3000
Open MetaMask wallet

## Running the tests

```bash
truffle test
```
